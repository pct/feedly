# Feedly app
simple feedly app built with electron, just a iframe url.

# install
    $ npm i -g electron-packger
    $ git clone https://github.com/pct/feedly
    $ cd feedly
    $ npm i
    $ electron-packager .

# Another way to create simple website app

    $ npm i -g nativefier
    $ nativefier feedly.com
